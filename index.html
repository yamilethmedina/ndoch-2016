Income Requirements

  • Section 8: Extremely-Low to Very-Low (up to 60%)
  • Workforce Housing: Very-Low - Low (60% - 80%)
Source

Application Process for Workforce Housing

  1. Visit Property
  2. Ask for Availability (If not, put on waiting list)
  3. Application Fee $35-$100
  4. Documents: Check Stubs, Bank Statement, W2, Birth Certificate, Social Security
  5. Credit Card and Criminal Check
  6. If Approved, Must Pay Holding Fee of $200-$300 (Refundable)

*This process must be done for each property

*2-week turnaround
